# av1-dwb-nome-sobrenome-2bimestre

## Descrição do Projeto

Catálogo de séries de TV construído com HTML5, CSS3, Bootstrap 5 e JavaScript puro, consumindo a API pública TVMaze. A aplicação apresenta uma página de listagem de séries com busca e carregamento incremental, além de uma página de detalhes para cada série.

## Objetivo Acadêmico

Demonstrar o consumo de APIs REST em uma aplicação web moderna, com foco em boas práticas de desenvolvimento front-end e interface responsiva seguindo requisitos de disciplina de Desenvolvimento Web.

## Tecnologias Utilizadas

- HTML5
- CSS3
- Bootstrap 5
- JavaScript puro
- Fetch API
- Async/Await
- JSON

## API Utilizada

- TVMaze API
  - Listagem: `https://api.tvmaze.com/shows`
  - Detalhes: `https://api.tvmaze.com/shows/{id}`

## Estrutura de Pastas

- `index.html`
- `detalhes.html`
- `css/style.css`
- `js/script.js`
- `js/detalhes.js`
- `README.md`

## Funcionalidades

- Listagem de séries em cards responsivos
- Busca em tempo real por nome da série
- Carregamento incremental de conteúdo
- Página de detalhes com informações completas da série
- Feedback visual de carregamento com spinner
- Tratamento de erros com mensagens amigáveis
- Design dark moderno com glassmorphism e animações suaves

## Como Executar Localmente

1. Abra a pasta do projeto no seu editor de código.
2. Abra `index.html` em um navegador moderno.
3. Navegue pela lista de séries e clique em "Ver Detalhes" para acessar `detalhes.html`.

> A aplicação consome diretamente a API pública TVMaze, então é necessário estar conectado à internet.
